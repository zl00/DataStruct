1. 公式ASL = ∑⦅i=1,n⦆Pi*Ci
   1. ASL: average search length
   2. n: n个关键字
   3. Pi: 第i个关键字被查找的概率，一般设为1/n
   4. Ci: 找到第i个关键字需要进行比较的次数，即查找长度。
